Instructions for staring up:
1. Install pycharm
2. Close any existing projects.
3. Check out from version control.
4. Choose git.
5. Write the URL of the GitHub project.
6. Press clone.
7. Go to file settings, under project slt_webapp choose project interpreter.
8. Click on the wheel icon on the top right and choose show all.
9. Press on the plus sign and, if not already selected choose existing environment and make sure the path is the python.exe in venv/scripts.
10. Press OK and apply.
11. In the terminal go in to the project folder by writing "cd slt_webapp".
12. In the terminal write "py manage.py run_chat_server".
13. Open new terminal tab and repeat “11” step.
14. In the terminal write "py manage.py runserver".
15. Press on the blue link.
16. Now you can interact with the application.
